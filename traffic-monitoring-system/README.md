## Traffic Monitoring System (Object Detection)
 
**Tech Stack:** 

- YOLOv8
- OpenCV
- ByteTrack
- FastAPI 

**Data Sources:**
 
- UA-DETRAC dataset
- Traffic camera footage (YouTube)
- Cityscapes dataset 

**Key Features:** 

- Vehicle Detection
- Counting
- Speed Estimation
- Traffic Flow Analysis 

**Hardware:** 

- Ubuntu server
- RTX 3060/4060 (8GB VRAM sufficient)
